Overview Inbound text message example
 
The inbound text message example demonstrates receiving a text message and delivering it in real-time to your application with the VoiceShot IVR API.
 

This demo is designed to work with the Developer Text Message Demo v5.0 campaign in your VoiceShot account.

 
In this example, you send a text message to a demonstration number, and the text message is delivered in XML to your Web server.
 
  Description of Inbound Text Message Example Menu Options
• Configure Account
  Configure communication between your Web server or application and your VoiceShot account.
   
• Test Configuration
  Verifies that VoiceShot can contact your Web server or application.
   
• Receive Text Message
  Instructions for receiving a text message.
   
• Call Summary
  Displays text message received with Call Summary.
