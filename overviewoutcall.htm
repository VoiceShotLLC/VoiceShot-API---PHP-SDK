Overview Outbound voice call example
 
The outbound voice call example demonstrates making a call with the VoiceShot IVR API.
 
By default, this example sends the Developer TTS Demo v5.0 campaign from your VoiceShot account.
 
  Description of Outbound Example Menu Options
• Configure Account
  Configure communication between your Web server or application and your VoiceShot account.
   
• Test Configuration
  Verifies that your Web server or application can contact VoiceShot.
   
• Make Call
  Sends XML to VoiceShot to initiate a call.
   
• Call Summary
  Displays call results obtained with Call Summary.
