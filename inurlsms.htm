Configure Account Inbound text message example
 
  Authorize your Phone
In order to send or receive text messages during the free trial, you must authorize the phone you are using.  To do so:
 
• Text your VoiceShot username to (248) 809-1234
• You will receive a text message confirming your number is authorized.
 
Once you purchase a VoiceShot text messaging plan, this authorization process is no longer required.
 
  Configure Inbound Call Summary URL
A Call Summary is a collection of data about a text message sent to your application when a text message is received.  For example, the Call Summary contains the content of the text message.  To receive Call Summaries for this example, do the following:
 
• Log into your VoiceShot account.
• Open the text messaging campaign named Developer Text Message Demo v5.0.
•

From the menu bar, click Campaign --> Options

• Click the Developer tab.
• For the Inbound Call Summary URL, enter:
http://yourserver/installdir/summaryinsms.php
• For more information on Call Summary, click here.
 
In the URL above, replace "yourserver" with the Web address of your Web server and "installdir" with the directory in which you installed the example.
